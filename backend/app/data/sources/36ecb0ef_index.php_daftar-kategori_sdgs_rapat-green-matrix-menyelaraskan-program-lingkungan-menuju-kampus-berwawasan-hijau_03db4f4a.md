Rapat Green Matrix: Menyelaraskan Program Lingkungan Menuju Kampus Berwawasan Hijau

Jum’at, 30 Januari 2026

Para pimpinan Kampus UPI Cibiru menggelar Rapat Green Matrix sebagai bagian dari upaya strategis dalam mewujudkan kampus yang berwawasan lingkungan dan berkelanjutan. Kegiatan ini berlangsung di Ruang Cawalaloka dengan agenda utama membahas berbagai program pengelolaan lingkungan yang relevan untuk tahun akademik 2026.

Dalam rapat tersebut, para pimpinan membahas strategi serapan air, pengembangan dan pemeliharaan lingkungan hijau, serta perencanaan kegiatan yang akan dijalankan pada tahun 2026. Diskusi difokuskan pada langkah-langkah konkret untuk meningkatkan efektivitas program Green Matrix, termasuk upaya konservasi sumber daya, penataan ruang terbuka hijau, serta pemanfaatan area kampus secara berkelanjutan.

Rapat ini menjadi wujud komitmen institusi dalam menyelaraskan kebijakan dan program lingkungan dengan prinsip keberlanjutan, sekaligus mendorong terciptanya kampus yang ramah lingkungan dan mampu memberikan contoh praktik terbaik bagi sivitas akademika. Dengan perencanaan matang dan partisipasi aktif seluruh pimpinan, diharapkan program Green Matrix dapat berjalan efektif dan berdampak positif pada pengelolaan lingkungan kampus di tahun 2026.
