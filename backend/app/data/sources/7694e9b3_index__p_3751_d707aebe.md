Program Studi S1 PSTI – UPI Kampus di Purwakarta

MikroTik Academy merupakan program kerja sama resmi yang ditawarkan oleh SIA MikroTik Latvia. Program ini bertujuan untuk membekali mahasiswa dengan kompetensi praktis di bidang jaringan komputer. Melalui integrasi materi dan kurikulum MikroTik ke dalam proses pembelajaran, program ini memberikan kesempatan kepada mahasiswa untuk memperoleh pengetahuan teknis serta sertifikasi internasional MikroTik Certified Network Associate (MTCNA).

Dengan menjadi bagian dari MikroTik Academy, PSTI dapat menyelenggarakan pelatihan dan ujian sertifikasi MTCNA secara mandiri melalui dosen yang telah tersertifikasi sebagai Academy Trainer. Program ini mendukung pembelajaran berbasis praktik dengan penggunaan langsung perangkat MikroTik serta sistem operasi RouterOS dalam laboratorium jaringan. Selain itu, program juga diharapkan dapat memfasilitasi lulusan PSTI yang unggul dan adaptif terhadap perkembangan global teknologi jaringan
