Program Studi Pendidikan Multimedia UPI Cibiru Laksanakan Program Student Mobility dengan Universiti Malaya Kelantan Tahun 2024

Cibiru, 24 Juli 2024 — Program Studi Pendidikan Multimedia Universitas Pendidikan Indonesia (UPI) Kampus Cibiru kembali menunjukkan komitmennya dalam memperluas pengalaman akademik mahasiswa melalui kerjasama internasional. Pada tahun 2024, program Student Mobility resmi dilaksanakan bekerja sama dengan Universiti Malaya Kelantan, Malaysia.

Program ini memberikan kesempatan bagi mahasiswa Pendidikan Multimedia UPI Cibiru untuk mengikuti kegiatan akademik dan budaya di Universiti Malaya Kelantan. Sebagai bagian dari inisiatif kerjasama global, program ini dirancang untuk meningkatkan pemahaman mahasiswa tentang pendidikan dan teknologi multimedia di tingkat internasional.

Selama program berlangsung, mahasiswa tidak hanya akan terlibat dalam perkuliahan dan proyek akademik, tetapi juga berpartisipasi dalam kegiatan budaya yang memperkuat hubungan antarbangsa. Selain itu, program ini memungkinkan pertukaran ide, inovasi, dan praktik terbaik di bidang multimedia, yang diharapkan dapat memperkaya pengalaman belajar para peserta.

Program ini diharapkan dapat terus berlanjut di tahun-tahun mendatang dan menjadi platform bagi mahasiswa untuk berkolaborasi di tingkat internasional, serta membuka peluang lebih luas dalam karier dan pengembangan kompetensi mereka di era globalisasi.
