PERINGATAN HARI BANDUNG LAUTAN API
“Kita tidak pernah dapat berkorban begitu besar bagi mereka yang bersedia mengorbankan segalanya bagi kita” – Mohammad Hatta. Peristiwa Bandung Lautan Api terjadi pada 24 Maret 1946. Peristiwa tersebut melibatkan tentara Indonesia, laskar pejuang, dan rakyat Bandung. Mereka membakar bangunan dan rumah mereka sendiri karena tidak rela daerahnya diduduki tentara Sekutu. Kami segenap keluarga besar HMP Penjas Kabinet Bahtera Abhipraya Mahasiswa PGSD Penjas UPI Sumedang mengucapkan selamat memperingati Hari Bandung Lautan Api.
