Software Engineering Study Program – UPI Campus in Cibiru.

The Software Engineering Undergraduate Program is one of the study programs at the UPI Campus in Cibiru, located on Jl. Raya Cibiru KM 15, Bandung Regency. In the Software Engineering Study Program, students will study fields related to Software Engineering (software development, software management, engineering processes, engineering tools and methodologies), Data Engineering (data modeling, knowledge-based data management, data management implementation), Multimedia & Game Programming (Game Development, 2D & 3D animation), and Ubiquitous Computing (Internet of Things, Augmented Reality, Cloud Computing).
