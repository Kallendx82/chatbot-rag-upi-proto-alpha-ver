# Sejarah Prodi Inpar

> Source: https://kd-sumedang.upi.edu/inpar/sejarah

INPAR UPI SUMEDANG
SEJARAH PROGRAM STUDI INDUSTRI PARIWISATA KAMPUS UPI DI SUMEDANG

Program Studi Industri Pariwisata UPI Kampus Sumedang resmi berdiri pada 23 Juli 2020. Berdirinya prodi ini merupakan langkah strategis untuk mendukung pengembangan destinasi wisata terutama di wilayah Sumedang serta meningkatkan kualitas SDM pariwisata di Indonesia.
