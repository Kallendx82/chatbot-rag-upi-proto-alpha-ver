Rapat Koordinasi Persiapan Program Tutorial SPAI Semester Genap Tahun 2025 di Kampus UPI Cibiru

UPI Cibiru - Kamis, 23 Januari 2025, Kampus Universitas Pendidikan Indonesia (UPI) di Cibiru menyelenggarakan rapat koordinasi persiapan program Tutorial Studi Pendidikan Agama Islam (SPAI) untuk semester genap tahun 2025. Kegiatan ini berlangsung di Tutorial Centre Kampus UPI Cibiru dengan dihadiri oleh Ketua Umum Tutorial terpilih, Asya Amelia Putri (mahasiswa Program Studi PMM, semester IV), Pembina Tutorial, Dr. Arief Hidayat, M.Pd.I, serta Kepala Tutorial Centre, Dr. Asep Rudi Nurjaman, M.Pd.I.

Rapat tersebut bertujuan untuk membahas persiapan teknis pelaksanaan program Binder (Bina Kader) sebagai bagian dari rangkaian program Tutorial SPAI yang dirancang untuk mensukseskan kegiatan tutorial pada semester genap mendatang. Program ini diharapkan dapat memperkuat kapasitas kader tutorial dalam mendukung pembinaan keagamaan yang lebih efektif di lingkungan Kampus UPI Cibiru.

Kegiatan ini diakhiri dengan sesi foto bersama yang diikuti oleh Ketua Umum Tutorial terpilih, Pembina Tutorial, dan Kepala Tutorial Centre sebagai bentuk simbolisasi semangat kolaborasi dan komitmen untuk keberhasilan program. Dengan adanya koordinasi yang matang, diharapkan pelaksanaan Tutorial SPAI semester genap 2025 dapat berjalan lancar, memberikan manfaat, serta mendukung pembinaan karakter mahasiswa di Kampus UPI Cibiru. Ibud. 23/1/2025.
